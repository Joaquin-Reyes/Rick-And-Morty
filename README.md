# 🧪 Rick and Morty Explorer

Aplicación web responsive que consume la API de Rick and Morty para visualizar y filtrar personajes.

## 🚀 Funcionalidades
- Listado de personajes desde API externa
- Filtro por género (masculino / femenino)
- Paginación dinámica
- Interfaz responsive adaptable a distintos dispositivos

## 🛠️ Tecnologías
- HTML
- CSS
- JavaScript
- API REST (Rick and Morty API)

## ⚙️ Características
- Consumo de API en tiempo real
- Renderizado dinámico de datos
- Navegación entre páginas
- Experiencia de usuario optimizada
